## Airapi scaffold

----

Using:

```shell
$ npm install -g aircode-node
$ aircode-node -h
$ aircode-node new hello
$ aircode-node generate book title:String price:Number
```

after new a project, you can use docker or node to start.

Running:

- Docker
```shell
$ cd hello
$ make build
$ make run
```

- Node monitor
```shell
$ cd hello
$ npm i
$ npm run dev
```
